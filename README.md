# AuthNative
React Native Facebook Login with Native Module

Setup

Go to the android/ directory of your react-native project
Create a file named local.properties with this line:
    sdk.dir = /Users/USERNAME/Library/Android/sdk
If your on windows:
    sdk.dir = C:\\Users\\USERNAME\\AppData\\Local\\Android\\sdk 
Where USERNAME is your OSX username

Open up the strings.xml file located at  res/values. Change the fb_app_id  by replacing the 0â€™s with your Facebook app id.
